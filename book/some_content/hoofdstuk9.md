# Functionele functies  

Soms is het handiger en overzichtelijker als je in plaats van hele lappen code in je main programma korte regels hebt die verwijzen naar aparte functies.

De algemene syntax voor een functie:  
>\# parameterloze functie  
>def functienaam():  
>	\# hier komt de code die de functie uit moet voeren  

>\# functie met 1 parameter en een return waarde  
>def functienaam(parameter):  
>	\# hier komt de code die   


Je hebt eerder het spel Steen-Papier-Schaar gemaakt, daarin kun je een aantal fases onderscheiden in het spel:
1.	spelen van het spel (computer kiest random waarde en speler kiest random waarde, beiden worden getoond)
2.	bepalen wie het spel gewonnen heeft (keuze speler en keuze computer vergelijk je en je geeft dan de winnaar van het spel terug)
